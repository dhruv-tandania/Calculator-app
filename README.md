# 🧮 Calculator Web App

A simple, responsive, and user-friendly Calculator Web Application built using **HTML**, **CSS**, and **JavaScript**. This project performs basic arithmetic operations with a clean and modern interface, making it easy to use on both desktop and mobile devices.

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear (AC) functionality
- ⌫ Delete (Backspace) functionality
- 📱 Responsive design
- 🎨 Clean and modern user interface

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
Calculator/
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🎯 Purpose

This project was created to practice front-end web development concepts, improve JavaScript logic, and gain hands-on experience with DOM manipulation and event handling.
